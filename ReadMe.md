# CS 4042  Group Project

Data is sourced from: https://www.basketball-reference.com/

All data  for Regular Season games.

### Requirements:

- Python
- Ipykernel
- pandas
- numpy
- matplotlib
- scikit-learn

### Running Instructions

1. Do not change the relative locations of files / subdirectories when running main.ipynb.
2. Please check you have the above requirements installed, if an import error pops up you can check the requirements.txt file.
3. Run through the main.ipynb file either all at once or cell by cell.
4. The final cell in main.ipynb creates a run loop which prompts the user for input, to exit this loop simply enter 0.
